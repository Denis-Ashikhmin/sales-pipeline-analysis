# Sales Pipeline Analysis — SLA Methodology
## Based on HubSpot Sales Enablement Framework

### Objective
Analyze B2B sales pipeline data to calculate key RevOps metrics
and build a Sales & Marketing SLA using HubSpot methodology.

### Key Metrics
| Metric | Value |
|--------|-------|
| Lead-to-Opportunity Rate | 94% |
| Opportunity-to-Close Rate | 51% |
| Average Deal Size | $2,360.91 |

### SLA Calculation
Assuming monthly revenue goal of $500,000:
- Deals needed per month: 212
- Opportunities needed: 416
- Qualified leads needed: 442

### Tools Used
SQL (SQLite) · Google Sheets · HubSpot SLA Framework

Maven Analytics — CRM Sales Opportunities Dataset
